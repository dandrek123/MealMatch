# MealMatch

MealMatch is a full-stack web application built with PHP and MySQL that helps users discover and manage meals through an interactive matching platform.

## Features

- User registration and login
- Session-based authentication
- Meal matching system
- Admin dashboard
- Analytics tracking
- Database-driven application

## Technologies

- PHP
- MySQL
- SQL
- HTML
- CSS
- JavaScript

## Project Structure

- `index.php` - Home page
- `login.php` - User authentication
- `register.php` - User registration
- `match.php` - Meal matching system
- `admin.php` - Administrative tools
- `analytics.php` - Analytics dashboard
- `db.php` - Database connection
- `schema.sql` - Database schema

## Setup

1. Import `schema.sql` into MySQL.
2. Configure database settings in `db.php`.
3. Run the project on a PHP-enabled web server.
